moTimeline

Lightweight, dependency-free JavaScript library for responsive two-column timeline layouts.

No jQuery No dependencies ESM · CJS · UMD MIT License < 3 kB gzipped
Get started View on GitHub

Live demo

    Get started

    npm

    npm install motimeline

    ESM import

    import MoTimeline from 'motimeline';
import 'motimeline/dist/moTimeline.css';

const tl = new MoTimeline('#my-timeline', {
  badgeShow: true,
});

    UMD (script tag, no bundler)

    <link rel="stylesheet" href="moTimeline.css">
<script src="moTimeline.umd.js"></script>
<script>
  const tl = new MoTimeline('#my-timeline');
</script>

    HTML structure

    <ul id="my-timeline">
  <li> <!-- your card content here --> </li>
  <li> <!-- ... -->               </li>
</ul>

    API

    // Initialize
const tl = new MoTimeline(elementOrSelector, options);

// Re-layout (e.g. after content changes)
tl.refresh();

// Initialize newly appended <li> items
tl.initNewItems();

// Tear down (removes listeners, resets DOM)
tl.destroy();

    Options

    Option Type Default Description
    columnCount object {xs:1, sm:2, md:2, lg:2} Number of columns at each breakpoint. Set a breakpoint to 1 to force single-column.
    badgeShow boolean false Show a numbered badge on the center line for each item.
    arrowShow boolean false Show an arrow indicator pointing from the item toward the center line.
