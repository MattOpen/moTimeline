moTimeline

Responsive two-column timeline — plain JavaScript, no dependencies

  • Project kickoff

    January 2024

    Kicked off the new moTimeline rewrite. Removed jQuery, migrated to plain JavaScript with a class-based API.

  • Vite build pipeline

    February 2024

    Set up Vite in library mode. Now ships ESM, CJS and UMD bundles so any project can import it.

  • Two-column layout

    March 2024

    The core layout algorithm places items in left and right columns automatically based on available height.

  • Responsive design

    April 2024

    Collapses to a single column on mobile. Breakpoints are fully configurable via columnCount.

  • Badge support

    May 2024

    Optional numbered badges appear on the center line. Enable with badgeShow: true.

  • GitHub Pages

    June 2024

    Published a live demo and documentation page using GitHub Pages. This is that page!
